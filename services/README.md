# Altus API services

The AI gateway moved to the sibling Go project:

```
../ide_apis/
```

Local dev: run `.\scripts\run-all.ps1` from the IDE repo (builds and starts `ide_apis/bin/gateway.exe` on port 8787).

Configuration: `../ide_apis/.env.local` (copy from `../ide_apis/.env.example`).

Production: `https://chat.altuside.com/v1`
